Voici le tout premier Trilux Collector! Son fonctionnement est très simple, il vous suffit de
remplir les cases d'informations et de veiller à sauvegarder vos données. Il vous est aussi
possible d'ouvrir un des fichiers textes sauvegardés dans Fichier/Ouvrir. Gardez en tête que
pour faciliter l'utilisation des autres contribueurs, qu'il est fortement recommandé de conserver
les fichiers textes dans le même répertoire. Merci d'avoir lu et bonne utilisation!
